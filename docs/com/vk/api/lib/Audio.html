add()method
public static function add(aid:String, oid:String, gid:String = ""):URLRequest

Копирует аудиозапись на страницу пользователя или группы.

Parameters
aid:String — id аудиозаписи.
 
oid:String — id владельца аудиозаписи. Если копируемая аудиозапись находится на странице группы, в этом параметре должно стоять значение, равное -id группы.
 
gid:String (default = "") — id группы, в которую следует копировать аудиозапись. Если параметр не указан, аудиозапись копируется не в группу, а на страницу текущего пользователя. Если аудиозапись все же копируется в группу, у текущего пользователя должны быть права на эту операцию.

Returns
URLRequest

get()method 
public static function get(uid:String = "", gid:String = "", aids:Array = null, need_user:Boolean = false):URLRequest

Возвращает список аудиозаписей пользователя или группы.

Parameters
uid:String (default = "") — id пользователя, которому принадлежат аудиозаписи. По умолчанию id текущего пользователя.
 
gid:String (default = "") — id группы, которой принадлежат аудиозаписи. Если указан параметр gid, uid игнорируется.
 
aids:Array (default = null) — список id аудиозаписей, входящие в выборку по uid или gid.
 
need_user:Boolean (default = false) — если этот параметр равен true, сервер возвратит базовую информацию о владельце аудиозаписей в структуре user (id, photo, name, name_gen).

Returns
URLRequest

reorder()method 
public static function reorder(aid:String, after:String, before:String, oid:String = ""):URLRequest

Изменяет порядок аудиозаписи, перенося ее между аудиозаписями, идентификаторы которых переданы параметрами after и before.

Parameters
aid:String — id аудиозаписи, порядок которой изменяется.
 
after:String — id аудиозаписи, после которой нужно поместить аудиозапись. Если аудиозапись переносится в начало, параметр может быть равен нулю.
 
before:String — id аудиозаписи, перед которой нужно поместить аудиозапись. Если аудиозапись переносится в конец, параметр может быть равен нулю.
 
oid:String (default = "") — id владельца изменяемой аудиозаписи. По умолчанию - id текущего пользователя.

Returns
URLRequest

search()method 
public static function search(q:String, sort:String, lirics:Boolean = false, count:String = "30", offset:String = "0"):URLRequest

Возвращает список аудиозаписей в соответствии с заданным критерием поиска.

Parameters
q:String — строка поискового запроса. Например, The Beatles.
 
sort:String — Вид сортировки. '1' - по длительности аудиозаписи, '0' - по дате добавления.
 
lirics:Boolean (default = false) — Если этот параметр равен true, поиск будет производиться только по тем аудиозаписям, которые содержат тексты.
 
count:String (default = "30") — количество возвращаемых аудиозаписей (максимум 30).
 
offset:String (default = "0") — смещение относительно первой найденной аудиозаписи для выборки определенного подмножества.

Returns
URLRequest

   import flash.net.FileReference;

   var file: FileReference;

   function chooseFile(): void
   {
     var file: FileReference = new FileReference();
     file.addEventListener(Event.SELECT, onSelect);
     file.browse([new FileFilter("MP3 Files (mp3)", "mp3")]);
   }

   function onSelect(): void
   {
     file.addEventListener(Event.COMPLETE, onComplete);
     file.upload(API.Audio.upload(upload_url), "file");
   }
   
   // при динамической загрузке:
   import com.vk.api.API; 

   function onComplete(e: Event): void
   {
     var res: Object = API.JSON.deserialize(e.target.data);
     trace("server="+res.server);
     trace("audio="+res.audio);
     trace("hash=" +res.hash);
   }
