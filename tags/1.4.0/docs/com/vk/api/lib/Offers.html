search()method 
public static function search(count:String, text:String = null, age_from:String = null, age_to:String = null, city:String = null, country:String = null, sex:String = null, online:Boolean = false, photo:Boolean = false, status:String = null, politic:String = null, university:String = null, edu_form:String = null, edu_status:String = null, school:String = null, district:String = null, station:String = null, group:String = null, company:String = null, position:String = null, religion:String = null, interests:String = null, name:String = null):URLRequest

Возвращает предложение случайного пользователя в соответствии с выбранными фильтрами.

Возвращает объект offer, содержащий поля uid, name, photo, date, message, age, online, city_id и city_name.

Parameters
count:String — количество возвращаемых предложений.
 
text:String (default = null) — ключевые слова в тексте предложения.
 
age_from:String (default = null) — минимальный возраст владельца предложения.
 
age_to:String (default = null) — максимальный возраст владельца предложения.
 
city:String (default = null) — id города владельца предложения.
 
country:String (default = null) — id страны владельца предложения.
 
sex:String (default = null) — id пола владельца предложения.
 
online:Boolean (default = false) — онлайн-статус владельца предложения.
 
photo:Boolean (default = false) — наличие фотографии у владельца предложения.
 
status:String (default = null) — id семейного положения владельца предложения.
 
politic:String (default = null) — id политических взглядов владельца предложения.
 
university:String (default = null) — id университета владельца предложения.
 
edu_form:String (default = null) — id формы обучения в вузе владельца предложения.
 
edu_status:String (default = null) — id статуса обучения в вузе владельца предложения.
 
school:String (default = null) — id школы владельца предложения.
 
district:String (default = null) — id района, указанного владельцем предложения в Местах.
 
station:String (default = null) — id станции метро, указанной владельцем предложения в Местах.
 
group:String (default = null) — id группы, в которой должен состоять владелец предложения.
 
company:String (default = null) — строка - компания владельца предложения.
 
position:String (default = null) — строка - должность владельца предложения.
 
religion:String (default = null) — строка - религия владельца предложения.
 
interests:String (default = null) — ключевые слова в интересах владельца предложения.
 
name:String (default = null) — ключевые слова в имени владельца предложения.

Returns
URLRequest
