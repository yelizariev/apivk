savePost()method 
public static function savePost(wall_id:String, post_id:String = null, server:String = null, photo:String = null, hash:String = null, photo_id_user:String = null, photo_id_photo:String = null, message:String = null):URLRequest

Сохраняет запись на стене пользователя.

Запись может содержать фотографию, ранее загруженную на сервер ВКонтакте или любую доступную фотографию из альбома пользователя.

Parameters
wall_id:String — id пользователя, на стене которого размещается запись.
 
post_id:String (default = null) — id записи, содержащий символы от a до z и от 0 до 9. Этот параметр будет передаваться в приложение через flashVars при просмотре или создании записи на стене пользователя.
 
server:String (default = null) — параметр, возвращаемый в результате загрузки изображения на сервер.
 
photo:String (default = null) — параметр, возвращаемый в результате загрузки изображения на сервер.
 
hash:String (default = null) — параметр, возвращаемый в результате загрузки изображения на сервер.
 
photo_id_user:String (default = null) — id пользователя, разместившего фотографию. Используется только совместно с параметром photo_id_photo.
 
photo_id_photo:String (default = null) — id фотографии.
 
message:String (default = null) — текст сообщения для размещения на стене пользователя.

Returns
URLRequest
