getProfiles()method 
public static function getProfiles(uids:Array, fields:Array = null, name_case:String = ""):URLRequest

Возвращает расширенную информацию о пользователях.

Parameters
uids:Array — список ID пользователей (максимум 1000 штук).
 
fields:Array (default = null) — поля анкет, необходимые для получения. Доступные значения: uid, first_name, last_name, nickname, sex, bdate (birthdate), city, country, timezone, photo, photo_medium, photo_big, has_mobile, rate.

Значения uid, first_name и last_name возвращаются всегда, вне зависимости от выбранных полей и их количества.

 
name_case:String (default = "") — падеж для склонения имени и фамилии пользователя. Возможные значения: именительный – nom, родительный – gen, дательный – dat, винительный – acc, творительный – ins, предложный – abl. По умолчанию nom.

Returns
URLRequest

getUserSettings()method 
public static function getUserSettings():URLRequest

Получает настройки текущего пользователя в данном приложении.

Возвращает битовую маску настроек текущего пользователя в данном приложени. Например, если метод возвращает 3, это означает, что пользователь разрешил отправлять ему уведомления и получать список его друзей.

  • +1 – пользователь разрешил отправлять ему уведомления.
  • +2 – доступ к друзьям.
  • +4 – доступ к фотографиям.
  • +8 – доступ к аудиозаписям.
  • +32 – доступ к предложениям.
  • +64 – доступ к вопросам.
  • +128 – доступ к wiki-страницам.
  • +256 – доступ к меню слева.

Returns
URLRequest
