get()method 
public static function get(uids:Array, qid:String = null, sort:String = "0", need_profiles:String = "0", count:String = "1", offset:String = "0"):URLRequest

Возвращает список вопросов, созданных пользователем.

Возвращает массив объектов - вопросов текущего пользователя, каждый из которых содержит поля qid, uid, type, text, answers_num, last_poster_id, last_poster_name, last_post_date, date, а также поля name, photo и online (если значение need_profiles в запросе больше 0).

Parameters
uids:Array — id пользователей, которым принадлежат вопросы, через запятую.
 
qid:String (default = null) — id отдельного вопроса, информацию о котором нужно получить. Если указан qid, то uids не учитывается.
 
sort:String (default = "0") — сортировка результатов (0 - по дате обновления в порядке убывания, 1 - по дате создания в порядке возрастания, 2 - по дате создания в порядке убывания).
 
need_profiles:String (default = "0") — определяет, требуется ли в ответе краткая информация об авторе вопроса (поля name, photo и online). Значения от 0 до 3. Чем больше значение, тем крупнее фотография в поле photo.
 
count:String (default = "1") — падеж для склонения имени и фамилии пользователя. Возможные значения: именительный – nom, родительный – gen, дательный – dat, винительный – acc, творительный – ins, предложный – abl. По умолчанию nom.
 
offset:String (default = "0") — количество вопросов, которое необходимо получить.

Returns
URLRequest

getAnswers()method 
public static function getAnswers(qid:String, sort:String = "0", need_profiles:String = "0", count:String = "1", offset:String = "0"):URLRequest

Возвращает список ответов на вопрос.

Возвращает количество найденных ответов и массив соответствующих им объектов, каждый из которых содержит поля aid, qid, uid, text, num, date, а также поля name, photo и online (если значение need_profiles в запросе больше 0).

Parameters
qid:String — id вопроса.
 
sort:String (default = "0") — сортировка результатов (1 - в порядке убывания даты, по умолчанию - в порядке возрастания)
 
need_profiles:String (default = "0") — определяет, требуется ли в ответе краткая информация об авторе ответа (поля name, photo и online). Значения от 0 до 3. Чем больше значение, тем крупнее фотография в поле photo.
 
count:String (default = "1") — количество ответов, которое необходимо получить.
 
offset:String (default = "0") — смещение, необходимое для выборки определенного подмножества ответов.

Returns
URLRequest
