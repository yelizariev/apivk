public static function getValues(language:String = null, keys:Array = null, all:Boolean = false):URLRequest

Возвращает список переведенных на указанный язык фраз.

Parameters
language:String (default = null) — ID языка, на котором необходимо получить переведенные фразы. По умолчанию выбирается язык текущего пользователя приложения.
 
keys:Array (default = null) — список ключей языковых фраз, перевод для которых необходимо получить.
 
all:Boolean (default = false) — если этот параметр равен true, то возвращается список из всех созданных фраз.

Returns
URLRequest
