init()method 
public static function init(api_url:String, viewer_id:String, api_id:String, secret:String, format:String, isTestMode:Boolean = false):void

Инициализация.

Необходимо сделать вызов данной прежде, чем обращаться каким-либо другим методам библиотеки apivk.

Если isTestMode задаётся значение false, то в качестве viewer_id нужно задавать id владельца приложения api_id.

Parameters
api_url:String — адрес сервиса API, по которому необходимо осуществлять запросы
 
viewer_id:String — id текущего пользователя, переданный SWF посредством flashvars при инициализации.
 
api_id:String — идентификатор приложения, присваивается при создании.
 
secret:String — секрет приложения.
 
format:String — формат возвращаемых данных
 
isTestMode:Boolean (default = false) — если этот параметр равен true, разрешает тестовые запросы к данным приложения. При этом аутентификация не проводится и считается, что текущий пользователь – это автор приложения. Это позволяет тестировать приложение без загрузки его на сайт. По умолчанию false.
